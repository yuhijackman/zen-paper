## UI
- "Cozy Soft-Minimalism" is the core aesthetic. The app should feel warm, personal, and inviting, moving away from cold corporate utility.
- Key Adjectives: Playful, Rounded, Airy, Tactile, Clean.
- Visual Metaphor: A well-organized, colorful storybook.

## Visual Rules and Guidelines
- Use soft, muted colors
- Use rounded corners
- Use a clean, modern font
- Use a consistent color scheme
- Use a consistent layout'
